\# Risk Management Agent

A standalone risk management agent for financial trading experiments.  

# Citation
The code and data in this repository are for the paper:
A. Wali, L. Hadrich Belguith, J.M. Alonso-Moral, "A Risk Management Agent with Uncertainty Quantification for Financial Trading", International Conference on Information Processing and Management of Uncertainty in Knowledge-Based Systems (IPMU 2026), https://doi.org/10.1007/978-3-032-28997-1_3

The current version supports OHLCV data loading, quantile-based risk estimation, and Gym-compatible risk policy evaluation.
